Deploy in 3 Steps

From code to production in minutes

1️⃣

Provision Infrastructure

Run Terraform workflow on GitHub Actions. Specify AWS credentials, region, and instance count. EC2 instances launch automatically.

2️⃣

Configure with Ansible

Trigger Ansible playbook with server IP. Nginx installs, SSL configures, and your site deploys. All automated and idempotent.

3️⃣

Access Your Site

Visit your server's public IP in any browser. Your application is live, monitored, and ready for production traffic.

The Bench 01 Stack

Production-grade technologies for reliable deployments

Frontend

Nginx: High-performance reverse proxy and web server. Handles static assets, SSL termination, and load balancing with minimal resource overhead.

Infrastructure

Terraform: AWS EC2 instances, security groups, and networking. Infrastructure versioning, state management, and disaster recovery built-in.

Orchestration

Ansible: Agentless configuration management. Playbooks define desired state; Ansible ensures compliance without system agents.

Automation

GitHub Actions: CI/CD workflows triggered on push. Terraform and Ansible runs orchestrated from version control.
