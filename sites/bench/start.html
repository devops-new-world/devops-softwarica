The Bench 01 Stack

Enterprise-grade technologies for reliable, scalable deployments

🌐 Frontend Layer

Nginx: High-performance reverse proxy and web server. Handles static assets, SSL termination, and load balancing with minimal resource overhead. Production-tested.

☁️ Infrastructure Layer

Terraform on AWS: EC2 instances, security groups, VPCs, and networking. Infrastructure versioning, state management, and disaster recovery all built-in.

🔧 Orchestration Layer

Ansible: Agentless configuration management. Playbooks define desired state; Ansible ensures compliance without system agents or complexity.

⚡ Automation Layer

GitHub Actions: CI/CD workflows triggered on push. Terraform and Ansible runs orchestrated from version control with full history and auditability.
